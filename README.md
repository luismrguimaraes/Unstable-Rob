# >>Unstɐbʅǝ_ꓤob<<

Developed in 48 hours for the RetroJam 2024 hosted by the IEEE UP Student Branch.

Theme: Unstable

## Description

In this platformer, you are a robot that heats up over time. Reach the cooling tanks quickly, or else...
